# WebCrawler
A Python Web Crawler, inspired by https://github.com/buckyroberts/Spider

Edit value of following variables in main.py

    PROJECT_NAME = ''
    HOMEPAGE = ''
    NUMBER_OF_THREADS = 8
    
Note : `NUMBER_OF_THREAD` must be an integer dataType .

To run program, use following code:
  
    python main.py
  
  
#### Built on Python3
